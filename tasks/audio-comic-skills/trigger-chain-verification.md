## 六、触发时序

### 6.1 正常流程

```
T0: 用户触发 "开始创作有声漫画"
T1: S3 接收并解析输入
T2: S1/S2/S4 准备资源
T3: 流水线第1环节开始
T4: S5 checkpoint 验收
T5: 流水线第2环节开始
...
Tn: 流水线第7环节完成
Tn+1: S6 良品率分析
Tn+2: S0 更新任务书
```

### 6.2 异常流程

```
T0: 用户触发
T1: S3 接收
T2: 环节执行
T3: S5 验收失败 → 漂移检测
T4: 超过阈值 → 回滚到 checkpoint
T5: 重新派发或升级模型
T6: 继续执行
```
