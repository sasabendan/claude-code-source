## 角色一致性管理策略

### 定位
S2 comic-style-consistency 的核心：同一个角色在不同分镜中视觉上保持一致。

### 角色注册表（必须字段）
```
## 角色注册表

| 角色 | 外貌特征 | 配色 | 声音描述 | 风格标签 |
|------|---------|------|---------|---------|
| 张三 | 短发、国字脸、军人气质 | 深蓝+金色 | 沉稳有力 | military, serious |
```

### 一致性方法

| 方法 | 说明 | 适用场景 |
|------|------|---------|
| 参考图（Reference Image）| 上传角色图作为生图参考 | MiniMax subject_ref, OpenAI ref |
| 风格标签 | 在 prompt 中加角色专属标签 | 所有服务商 |
| 配色方案 | 固定角色主色调 | 生图 prompt 固定前缀 |
| 固定描述词 | 每次生图 prompt 必须包含的角色描述 | 防止风格漂移 |

### 参考图管理
- 参考图池：`knowledge-base/characters/<角色名>/ref.png`
- 每次生图前：先读取参考图路径，拼接进 prompt
