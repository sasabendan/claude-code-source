## NCA 三步必要条件筛选法

### 定位
用"底线思维"验证每个 pipeline 环节，不接受"平均分"，只接受"及格线"。

### 三步标准（同时满足）

1. **约束力够强**：d > 0.1（效应量 ≥ 中等）
2. **模型够准**：c-accuracy > 95%
3. **统计上显著**：p < 0.05

### 对本项目验收的映射
- S4 监督的每个 checkpoint = 一次 NCA 检查
- "及格线" = 风格锚定参数达标 / 脚本字数误差 <10% / 场景完整率 100%
- 不是"平均表现如何"，而是"没有这个环节整条线是否废掉"

### 天花板线 vs 趋势线
- 充分性（回归）：趋势线 → X 增加 Y 平均变化多少
- 必要性（NCA）：天花板线 → X 低于阈值 Y 绝无可能
