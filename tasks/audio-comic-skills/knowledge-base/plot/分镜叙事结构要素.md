## 分镜叙事结构要素

### 定位
分镜环节（S3 第 2 步）的设计原理，从 baoyu 图表系统和叙事结构提取。

### 每个分镜必须包含

1. **场景 (Scene)**：地点、时间、光线
2. **人物 (Characters)**：谁在场，角色状态
3. **动作 (Action)**：发生了什么
4. **对话/旁白 (Dialogue/Narration)**：说了什么，想了什么
5. **节奏标记 (Pacing)**：这个分镜占多长时间

### 分镜描述格式模板
```
## 分镜 N：<场景名称>
- **地点**：<具体位置>
- **时间**：<白天/夜晚/具体时间>
- **人物**：<在场角色，状态>
- **画面描述**：<镜头角度、画面内容、构图要点>
- **对话**：<原文对话>
- **旁白**：<内心独白/叙述>
- **时长**：<估计秒数>
- **分镜提示词**：<给 AI 生图的 prompt 开头>
```

### 与生图 prompt 的关系
分镜描述 → 生成生图 prompt → 参考图册（角色一致性）→ 最终图片
