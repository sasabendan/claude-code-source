# chinese-thinking（中文思维锚定）

> 思考启动锚点，不是生产检验环节。
> 源码：`skills/chinese-thinking/SKILL.md`（v1.3，2026-05-01）

## 定位变更（v1.3）

| 本 Skill | 不是 |
|---------|------|
| 思考启动锚点 ✅ | 生产线检验环节 ❌ |
| 确认"交付什么" | 定义"怎么做" |
| 产出物导向 | 过程理解导向 |

**检验工作由 skill-change-test.sh 承担（机器验证，客观可靠）。本 Skill 只负责让每次执行从"交付什么"开始。**

## 三段式格式（v1.3）

```
【主线锚定】
任务要求产出：<具体产物描述（含文件路径/数量/格式）>
当前产出状态：【有/无/部分】
与目标的差距：<具体不足，或"已达标">
```

## 填写规范

| 字段 | 规范 |
|------|------|
| 任务要求产出 | 必须具体：含文件名、数量、格式 |
| 当前产出状态 | 三选一：【有】【无】【部分】 |
| 与目标的差距 | 若【有】→写"已达标"；若【无/部分】→写具体差距 |

## 示例

```
【主线锚定】
任务要求产出：skills/chinese-thinking/SKILL.md v1.3 + .index.jsonl updated 同步
当前产出状态：【部分】
与目标的差距：SKILL.md 已更新，.index.jsonl 待同步
```
