## Knowledge Comic Creator Skill

Claude Code 专业叙事引擎，用于从文本/源文档生成知识漫画。

### 原文位置
`reference-07-comic-creator-skill.md`

### 核心功能
1. 动态布局控制（标准漫画、Webtoon、电影分镜、技术图解）
2. 视觉风格库（Manga、Realistic、Wuxia、Sepia 等）
3. 多变体分镜（时间线/主题/角色叙事）
4. 后制工具（PDF 编译）
5. 角色一致性管理

### 使用场景
- 技术文档转漫画教程
- 传记/历史可视化
- 商业案例/培训材料
