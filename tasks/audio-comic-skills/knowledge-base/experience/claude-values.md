# claude-values（价值观与主任务理解）

> 判断「什么重要」「该不该做」时触发。
> 源码：`skills/claude-values/SKILL.md`

## 核心价值观：知识库价值层级

```
待用储备信息（尚未排布）
    ↓ 学习别人
可用信息（有价值但无本项目成功经验）
    ↓ 打磨自己
可变现经验/技能（可用于生产）
```

## 关键理解

- **信息排布产生知识**：双链信息本身可以产生新的信息组合（新知识）
- **知识库是生产能力的前提**：生产线是硬约束，知识库是优化前提
- **尚未成功≠无价值**：探索价值在于无人走通的长链路

## 主任务理解

**多AGI无监督自动化有声漫画生产**
- 尚未被证实成功（否则早有人发出来）
- 长生产链路，多AGI协作
- 统一管理框架多线程并发
