## 每日授权 + 记忆系统协议

### 定位
防止 Claude 遗忘关键信息（授权范围、密码）和越界行动。

### 核心原则
**主任务 = 边界，边界之外必须取得明确授权才能行动**

### 授权管理
- 当前授权范围：`~/.claude/authorized-scope.jsonl`
- 每日打卡：主任务未完成前，每天找 user 整体确认授权一次
- 工作完成后，原授权全部暂停，存档

### 记忆系统
- `memory_store.py`：add/get/list/search/delete 核心记忆
- 存储：`~/.claude/memory-store.jsonl`（JSONL，每行一条）
- 密码：只存 macOS Keychain，不存云同步文件

### Keychain 操作
```bash
security add-generic-password -s "claude-backup" -w "密码"
security find-generic-password -s "claude-backup" -w
```
